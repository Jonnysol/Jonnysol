# Jonathan Solomon

I build practical systems at the edge of AI, spatial computing, robotics, IoT, and media.

Currently focused on tools that make computers understand physical space, speech, and messy real-world workflows.

## Featured Work

- [PrismLabs](https://github.com/Jonnysol/prismlabs-landing) - ambient OS concept for connecting software, sensors, and physical environments.
- [Movieport](https://github.com/Jonnysol/Movieport) - voice-controlled media OS with local Whisper transcription, content matching, and streaming workflows.
- [TimeCapsuleTV](https://github.com/Jonnysol/TimeCapsuleTV) - retro broadcast engine that turns curated video libraries into decade-based channel surfing.
- [DCESK8](https://github.com/Jonnysol/DCESK8) - community infrastructure for Washington DC electric skateboard and personal electric vehicle riders.
- [NASA Plant Prediction](https://github.com/Jonnysol/NASA-Based-Plant-Prediction-Software) - environmental intelligence toolkit using NASA-inspired data, crop pages, solar calculations, docs, and tests.
- [Amharic Voice AI](https://github.com/Jonnysol/Amharic-AI-Voice-Chatbot) - Amharic voice assistant prototype for Raspberry Pi, speech-to-text, LLM reasoning, and Ge'ez-script language access.

## Areas I Keep Coming Back To
